# Rahasia

Private text and file sharing app with encrypted storage and link-based sharing.

## Status

Early development.

## Planned Stack

- Frontend: React + Vite
- Backend: FastAPI
- Database: PostgreSQL

## Repository Structure

- `frontend/` frontend application
- `backend/` backend application
- `infra/` local infrastructure and deployment files
- `docs/public/` public documentation

## Goal

Build a simple app where a sender can share text or files with a recipient using a link plus a password or generated decryption key.
